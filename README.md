## Getting Started

### Installing

1. Clone the repo:

```bash
git clone https://github.com/alexdalene/alex_dalene_exam1.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run sass
```
Then use live code extension for visual studio.
